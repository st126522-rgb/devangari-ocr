# OCR-Devanagari-CRNN

Scaffold for OCR project using CRNN + LSTM.

Next steps:
- Install requirements: pip install -r requirements.txt
- Use data/utils/data_loader.py to load the HF dataset.
- Use TRDG (https://github.com/Belval/TextRecognitionDataGenerator) to synthesize images from text.
- Implement training in scripts/train.py using model/ocr_model.py.
